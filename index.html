Prospera Blockchain

Layer 1 Network Quantum-Resistant

Prospera adalah jaringan blockchain Layer 1 (L1) masa depan yang dirancang menggunakan bahasa pemrograman Rust untuk kecepatan maksimal dan sistem kriptografi pasca-kuantum guna menangkal ancaman komputer kuantum.

Fitur Utama Jaringan

  • Ketahanan Kuantum: Mengamankan aset dari serangan komputer kuantum masa depan.
  • Performa Tinggi Rust: Pemrosesan transaksi cepat, hemat memori, dan bebas garbage collection.
  • Open Source & Transparan: Seluruh kode inti dapat diaudit langsung oleh publik di GitHub.

Cara Menjalankan Node (Via Termux / Linux)

Pastikan Anda sudah menginstal alat kompilasi Rust, lalu jalankan perintah berikut:

git clone https://github.com
cd prospera-blockchain
cargo run
